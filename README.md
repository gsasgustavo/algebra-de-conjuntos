# Álgebra de Conjuntos

Operações da Álgebra de Conjuntos

Crie um (ou mais) programa(s) (utilizando a linguagem ou plataforma de programação de sua preferência) que implementa as operações da álgebra de conjuntos.

Crie um documento que apresenta, para cada operação da álgebra de conjuntos, o algoritmo (utilizando notação de pseudo-código), a sua explicação e uma demonstração. Para a demonstração, apresente trechos de código que utilizam a implementação do item anterior. Exemplo:

> A = {1,2,3}  
> B = {3,2,4}  
> C = uniao(A, B)

(apresente/imprima o conjunto resultante).
